# AutoVScodeCEnvironment
自动配置C语言开发环境

这个程序可以帮助萌新们自动配置C语言环境，自动安装MinGW编译器（安装目录
**C:\\MinGW**
），自动设置环境变量，自动安装C/C++插件

**一键化配置开发环境，你值得拥有**

# FAQ

### 为什么安装C/C++失败？
请在安装过VS code后重启电脑，若仍然无法安装，请提交Issue。

### 为什么自动启动VS code，F5调试后出现未知命令错误？
请重启VS code。

### 为什么点击安装后窗口闪一下就没了？
请解压后再运行程序，切勿在压缩包内运行。解压出来的DLL文件请不要删除。
